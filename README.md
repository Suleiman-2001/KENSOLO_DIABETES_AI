## ⚕️ Features

- 🔍 **Clinical Data Detection** – Automatically detects missing values, outliers, and inconsistencies  
- 🧬 **Diabetes Prediction Engine** – Predicts risk levels using machine learning models  
- 📊 **Healthcare Analytics Dashboard** – Visualizes patient trends and health patterns  
- 🧠 **AI Recommendations** – Generates clinical decision support suggestions  
- 📝 **Data Summaries (NLP)** – Converts raw medical data into readable insights  
- 💾 **Export Reports** – Download results in CSV, Excel, and PDF formats
